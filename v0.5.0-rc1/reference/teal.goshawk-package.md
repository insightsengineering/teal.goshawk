# teal.goshawk core packages

The teal.goshawk package renders the UI and calls the respective
biomarker visualization functions.

## Details

The data used for teal.goshawk have some constraints. It must contain
the columns `AVISITCD`, `BASE`, `BASE2`, `AVALU`, `LBSTRESC`, `LOQFL`,
`CHG2`, and `PCHG2`.
